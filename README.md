# Picture To MIDI

*Max 8.5*

This Max Patch turns a picture (.jpg or .png) into a MIDI-file.

The controls are:<br/>
**Drop File**<br/>
**Invert** colors<br/>
**Brightness** of the picture<br/>
**Contrast** of the picture<br/>
**Convert** picture to MIDI-file<br/>
**Save** the MIDI-file<br/>
